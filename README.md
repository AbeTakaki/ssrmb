# ssrmb

`ssrmb` は、Windows のスクリーンショットフォルダを自動で整理する CLI ツールです。
Cobra フレームワークを採用しております。

## 特徴

- 🧹 **整理**: スクリーンショット画像を、ファイルの更新日に基づいて `yyyy-mm-dd` 形式のフォルダへ自動で振り分けます。
- 🤖 **クリーンアップ**: 指定した期間（月単位・日単位）より古いフォルダを判別して削除します。
- 🛡️ **その他機能**: 
  - 削除前には必ず対象一覧を表示し、ユーザーの確認（y/N）を求めます。
  - 同名ファイルがある場合は自動で連番（`_1`, `_2`...）を付与し、上書きを防止します。
- 🔍 **日付フォルダ限定**: `yyyy-mm-dd` 形式以外のフォルダ（例：「保存用」「旅行」など）は、対象外として処理を行います。

## インストール
### Go 環境がない場合（推奨）
1. [Releases](https://github.com/AbeTakaki/ssrmb/releases) ページから最新の `ssrmb.exe` をダウンロードします。
2. ダウンロードしたファイルを任意のフォルダに配置します。
3. コマンドプロンプトまたは PowerShell を開き、配置したフォルダへ移動して実行します。
```bash
.\ssrmb.exe
```
### ビルド
Go がインストールされている環境でビルドします。


```bash
git clone [https://github.com/yourusername/ssrmb.git](https://github.com/yourusername/ssrmb.git)
cd ssrmb
go mod tidy
go build -o ssrmb.exe main.go
```

## 使い方
```ssrmb``` を使用して整理・清掃を開始します。

### 1. 画像の整理のみを実行
フォルダ内の画像を日付ごとに仕分けます。

```bash
.\ssrmb.exe
```

### 古いフォルダを削除（クリーンアップ）
整理と同時に、指定期間より前のフォルダを削除します。

- 14日以上前のフォルダを消す場合:

```bash
.\ssrmb.exe --rmd 14
# 短縮形
.\ssrmb.exe -d 14
```

- 3ヶ月以上前のフォルダを消す場合:

```bash
.\ssrmb.exe --rmm 3
# 短縮形
.\ssrmb.exe -m 3
```

### 3. ヘルプの表示
利用可能なオプションを確認できます。

```bash
.\ssrmb.exe --help
```

## 🛠 コマンド体系

| フラグ | 短縮形 | 説明 |
| :--- | :--- | :--- |
| `--rmd` | `-d` | 指定した**日数**より前のフォルダを削除 |
| `--rmm` | `-m` | 指定した**月数**より前のフォルダを削除 |

## ⚠️ 免責事項
- クリーンアップ機能によって削除されたフォルダは、ゴミ箱を経由せず完全に削除されます。実行時の確認画面をよくご確認ください。
