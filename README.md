# Screenshot Rmb CLI

Windows の「スクリーンショット」フォルダを日付ごとに自動整理し、古いフォルダをクリーンアップするための CLI ツールです。

## ✨ 主な機能

- **自動振り分け**: `Pictures/Screenshots` 直下にある画像を、ファイルの更新日に基づいて `yyyy-mm-dd` 形式のフォルダへ移動します。
- **上書き防止**: 移動先に同名のファイルがある場合、自動的に連番（`_1`, `_2`...）を付与して保存します。
- **クリーンアップ**: 指定した期間（月数・日数）より前の日付フォルダを一括削除します。
- **安全設計**: 削除の実行前に、対象フォルダを一覧表示し、ユーザーの確認（y/n）を求めます。

## 🚀 使い方

### 1. 準備（ビルド）
Go がインストールされている環境で、以下のコマンドを実行してバイナリを作成します。

```bash
go build -o ssrmb.exe main.go
```

### 2. 実行
作成された ```ssrmb.exe``` を使用します。
**画像を整理するだけの場合**

```bash
.\ssrmb.exe
```

**1ヶ月以上前の古いフォルダを削除したい場合**

```bash
.\ssrmb.exe -rmm 1
```
**14日以上前の古いフォルダを削除したい場合**

```bash
.\ssrmb.exe -rmd 14
```

## 🛠 仕様
- **対象ファイル**: ```.png```, ```.jpg```, ```.jpeg```
- **対象パス**: 以下の順で自動探索します。

  1. ```%USERPROFILE%\Pictures\Screenshots```
  2. ```%USERPROFILE%\OneDrive\画像\Screenshots```
  3. ```%USERPROFILE%\OneDrive\Pictures\Screenshots```

- **削除対象**: ```yyyy-mm-dd``` 形式の名前を持つフォルダのみを判別して削除します。手動で作成した別名のフォルダは削除されません。

### ⚠️ 注意事項
- 削除機能（```-rmm```, ```-rmd```）によって削除されたフォルダは、**ゴミ箱に入らず完全に削除されます。**必ず実行時の確認メッセージを確認してください。
